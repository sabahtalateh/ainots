# Essence of Calculus (Основы математического анализа)

Конспект по плейлисту [3Blue1Brown - Essence of Calculus](https://youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr&si=1QXG9w-RWVlvyYVF)

Цель серии — не просто запомнить формулы, а понять их происхождение так глубоко, чтобы вы могли "изобрести" их сами.

Матанализ изучает **поведение функций при бесконечно малых изменениях**:

- **Производная** — как функция меняется локально (отношение малых изменений): $\frac{dy}{dx}$
- **Интеграл** — как накапливается эффект (сумма малых вкладов): $\int f(x)\,dx$
- **Фундаментальная теорема** — эти два взгляда связаны: накопление и скорость накопления — обратные операции
