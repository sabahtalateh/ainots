# Число π (пи)

## Что такое π?

$$\pi = \frac{\text{длина окружности}}{\text{диаметр}} = \frac{C}{2R} \approx 3.14159265...$$

Это **отношение** длины окружности к её диаметру — одинаковое для **всех** кругов.

## Откуда известно, что π существует?

### 1. Эмпирическое наблюдение (верёвка)

Древние люди:
- Оборачивали верёвку вокруг круглых предметов
- Измеряли длину верёвки и диаметр
- Замечали: отношение всегда **примерно одинаковое** (~3 с чем-то)

### 2. Геометрическая логика

**Все круги подобны** — это ключевое!

Если увеличить круг в 2 раза:
- Диаметр × 2
- Длина окружности × 2  
- Отношение C/d остаётся **тем же**

Значит, существует **универсальная константа** для всех кругов. Её назвали π.

## Почему верёвки было недостаточно?

| Проблема | Следствие |
|----------|-----------|
| Верёвка растягивается | Разные измерения: 3.1, 3.2, 3.14... |
| Круги неидеальны | Горшок ≠ математический круг |
| Точность 2-3 знака max | Для астрономии и навигации мало |
| Не отвечает на вопрос | π — дробь? корень? что это? |

## Главная проблема: нужен был алгебраический способ!

> **Ключевой вопрос:** Как вычислить π **без физического измерения**?

Верёвка даёт приближение, но:
- Зависит от качества инструментов
- Ограничена человеческой точностью
- Не даёт **формулу** для вычисления

### Что требовалось:

1. **Алгоритм** — последовательность шагов, дающая π с любой точностью
2. **Формула** — математическое выражение, равное π
3. **Независимость от измерений** — чистая математика, никаких верёвок!

### Прорыв Архимеда

Архимед нашёл **геометрический алгоритм**:
- Не нужна верёвка
- Только циркуль и линейка (для построения многоугольников)
- Можно вычислять сколь угодно точно (добавляя стороны)

### Прорыв XVII века — бесконечные ряды

$$\pi = 4 - \frac{4}{3} + \frac{4}{5} - \frac{4}{7} + \frac{4}{9} - ...$$

Это **чисто алгебраическая формула**:
- Никаких кругов, верёвок, измерений
- Только арифметика: сложение и деление
- Бесконечное количество знаков π

**Это был настоящий прорыв:** π можно вычислить, сидя за столом с бумагой!

## История вычисления π

| Метод | Кто/Когда | Точность |
|-------|-----------|----------|
| Верёвка | Древние цивилизации | π ≈ 3 |
| Вписанные многоугольники | Архимед (~250 до н.э.) | $3.1408 < \pi < 3.1429$ (96-угольник) |
| Многоугольники | Лю Хуэй (263 н.э.) | 5 знаков (3072-угольник) |
| Бесконечные ряды | Мадхава, Лейбниц (XIV-XVII вв.) | произвольная точность |
| Компьютеры | Современность | триллионы знаков |

### Метод Архимеда: многоугольники

Нельзя измерить кривую напрямую, но можно **приблизить**:

- Вписанный шестиугольник: периметр = 6r = 3d → **π > 3**
- Описанный шестиугольник: периметр > 6r → **π < 3.5**
- Больше сторон → точнее приближение

Длина окружности **зажата** между периметрами вписанного и описанного многоугольников.

### Бесконечные ряды

$$\pi = 4\left(1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \frac{1}{9} - ...\right)$$

Это ряд Лейбница — медленно сходится, но даёт **точное** значение π.

## Природа числа π

Математики доказали:

| Свойство | Кто доказал | Что значит |
|----------|-------------|------------|
| **Иррациональное** | Ламберт, 1761 | π ≠ дробь (22/7 — только приближение) |
| **Трансцендентное** | Линдеман, 1882 | π не корень никакого алгебраического уравнения |

Следствие: **невозможно построить квадрат с той же площадью, что у круга** (циркулем и линейкой).

## Зачем нужна точность?

| Задача | Достаточная точность |
|--------|---------------------|
| Построить колесо | π ≈ 3 |
| Построить храм | π ≈ 3.14 |
| Астрономия, навигация | 10+ знаков |
| GPS-спутники | 15+ знаков |
| Проверка алгоритмов | миллионы знаков |

## Связь с площадью круга

Формула площади круга:

$$S = \pi R^2$$

Эта формула **использует** π, но не **вычисляет** его. Чтобы знать площадь, нужно уже знать значение π.

### Геометрический вывод (развёртка в треугольник)

1. Разрежь диск на тонкие кольца
2. "Разверни" каждое кольцо в прямоугольник
3. Прямоугольники образуют треугольник:
   - Основание = $2\pi R$ (длина окружности)
   - Высота = $R$ (радиус)
4. Площадь = $\frac{1}{2} \cdot 2\pi R \cdot R = \pi R^2$

## Итог

- **Существование** π очевидно из геометрии (все круги подобны)
- **Приближённое значение** можно измерить верёвкой
- **Точное значение** требует математики (ряды, алгоритмы)
- **Природа** π (иррациональность, трансцендентность) — глубокий математический вопрос

Верёвка дала **приближение**. Математика дала **понимание**.
