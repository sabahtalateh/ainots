## –ó–∞–¥–∞—á–∞

–£–ø—Ä–æ—Å—Ç–∏—Ç—å –≤—ã—Ä–∞–∂–µ–Ω–∏–µ:

$$\frac{x^3 + x^2 - x - 1}{x^2 + 2x + 1}$$

## –†–µ—à–µ–Ω–∏–µ

### –®–∞–≥ 1: –§–∞–∫—Ç–æ—Ä–∏–∑—É–µ–º –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª—å

$$x^2 + 2x + 1 = (x+1)^2$$

–≠—Ç–æ –ø–æ–ª–Ω—ã–π –∫–≤–∞–¥—Ä–∞—Ç.

### –®–∞–≥ 2: –§–∞–∫—Ç–æ—Ä–∏–∑—É–µ–º —á–∏—Å–ª–∏—Ç–µ–ª—å

$$x^3 + x^2 - x - 1$$

–°–≥—Ä—É–ø–ø–∏—Ä—É–µ–º —á–ª–µ–Ω—ã:

$$= x^2(x+1) - 1(x+1)$$

–í—ã–Ω–µ—Å–µ–º –æ–±—â–∏–π –º–Ω–æ–∂–∏—Ç–µ–ª—å $(x+1)$:

$$= (x+1)(x^2-1)$$

–†–∞–∑–ª–æ–∂–∏–º —Ä–∞–∑–Ω–æ—Å—Ç—å –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ $x^2 - 1$:

$$= (x+1)(x-1)(x+1)$$

$$= (x-1)(x+1)^2$$

### –®–∞–≥ 3: –°–æ–∫—Ä–∞—â–∞–µ–º –¥—Ä–æ–±—å

$$\frac{x^3 + x^2 - x - 1}{x^2 + 2x + 1} = \frac{(x-1)(x+1)^2}{(x+1)^2} = x-1$$

## –û—Ç–≤–µ—Ç

$$x - 1, \quad x \neq -1$$

**–í–∞–∂–Ω–æ:** –ü—Ä–∏ $x = -1$ –∏—Å—Ö–æ–¥–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –Ω–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–æ (–¥–µ–ª–µ–Ω–∏–µ –Ω–∞ –Ω–æ–ª—å).

## –ü—Ä–æ–≤–µ—Ä–∫–∞ –≤ SymPy

```python
from sympy import symbols, simplify, factor, cancel

x = symbols('x')
expr = (x**3 + x**2 - x - 1) / (x**2 + 2*x + 1)

# –°–ø–æ—Å–æ–± 1: –£–ø—Ä–æ—â–µ–Ω–∏–µ
print(simplify(expr))  # x - 1

# –°–ø–æ—Å–æ–± 2: –°–æ–∫—Ä–∞—â–µ–Ω–∏–µ –¥—Ä–æ–±–∏
print(cancel(expr))    # x - 1

# –§–∞–∫—Ç–æ—Ä–∏–∑–∞—Ü–∏—è —á–∏—Å–ª–∏—Ç–µ–ª—è
print(factor(x**3 + x**2 - x - 1))  # (x - 1)*(x + 1)**2

# –§–∞–∫—Ç–æ—Ä–∏–∑–∞—Ü–∏—è –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª—è
print(factor(x**2 + 2*x + 1))       # (x + 1)**2
```

# –ú–µ—Ç–æ–¥ –ö–∞—Ä–¥–∞–Ω–æ –¥–ª—è –∫—É–±–∏—á–µ—Å–∫–∏—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π

## –í—ã–≤–æ–¥ —Ñ–æ—Ä–º—É–ª—ã –ö–∞—Ä–¥–∞–Ω–æ

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ $x^3 - 3x - q = 0$ –∏–ª–∏ $x^3 = 3x + q$.

### –ò–¥–µ—è –º–µ—Ç–æ–¥–∞

–ë—É–¥–µ–º –∏—Å–∫–∞—Ç—å —Ä–µ—à–µ–Ω–∏–µ –≤ –≤–∏–¥–µ **—Å—É–º–º—ã –¥–≤—É—Ö –Ω–µ–∏–∑–≤–µ—Å—Ç–Ω—ã—Ö**:

$$x = u + v$$

–≥–¥–µ $u$ –∏ $v$ ‚Äî –ø–æ–∫–∞ –Ω–µ–∏–∑–≤–µ—Å—Ç–Ω—ã–µ –≤–µ–ª–∏—á–∏–Ω—ã, –∫–æ—Ç–æ—Ä—ã–µ –º—ã –ø–æ–¥–±–µ—Ä–µ–º —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º –æ–±—Ä–∞–∑–æ–º.

### –®–∞–≥ 1: –†–∞—Å–∫—Ä—ã—Ç–∏–µ –∫—É–±–∞ —Å—É–º–º—ã

–ü–æ–¥—Å—Ç–∞–≤–∏–º $x = u + v$ –≤ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ. –í—Å–ø–æ–º–Ω–∏–º —Ñ–æ—Ä–º—É–ª—É –∫—É–±–∞ —Å—É–º–º—ã:

$$(u + v)^3 = u^3 + 3u^2v + 3uv^2 + v^3$$

–í—ã–Ω–µ—Å–µ–º $3uv$ –∑–∞ —Å–∫–æ–±–∫—É –∏–∑ —Å—Ä–µ–¥–Ω–∏—Ö —á–ª–µ–Ω–æ–≤:

$$(u + v)^3 = u^3 + v^3 + 3uv(u + v)$$

–ü–æ—Å–∫–æ–ª—å–∫—É $x = u + v$, –ø–æ–ª—É—á–∞–µ–º:

$$x^3 = u^3 + v^3 + 3uvx$$

### –®–∞–≥ 2: –ü–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫–∞ —Å–ª–∞–≥–∞–µ–º—ã—Ö

–ü–µ—Ä–µ–ø–∏—à–µ–º –≤ –≤–∏–¥–µ:

$$x^3 = 3uvx + (u^3 + v^3)$$

### –®–∞–≥ 3: –ú–µ—Ç–æ–¥ –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–≤ (—Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–≤)

–£ –Ω–∞—Å –µ—Å—Ç—å **–¥–≤–∞ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –¥–ª—è $x^3$**, –∫–æ—Ç–æ—Ä—ã–µ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Ä–∞–≤–Ω—ã:

**–ò—Å—Ö–æ–¥–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ:**
$$x^3 = 3x + q$$

**–ù–∞—à–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ (–ø–æ—Å–ª–µ –ø–æ–¥—Å—Ç–∞–Ω–æ–≤–∫–∏ $x = u + v$):**
$$x^3 = 3uvx + (u^3 + v^3)$$

#### –ü–æ–¥—Ä–æ–±–Ω–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ –º–µ—Ç–æ–¥–∞ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–≤

–ï—Å–ª–∏ –¥–≤–∞ –º–Ω–æ–≥–æ—á–ª–µ–Ω–∞ —Ä–∞–≤–Ω—ã **–ø—Ä–∏ –ª—é–±–æ–º** –∑–Ω–∞—á–µ–Ω–∏–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, —Ç–æ **–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã –ø—Ä–∏ –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö —Å—Ç–µ–ø–µ–Ω—è—Ö** –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Ä–∞–≤–Ω—ã.

**–ü–µ—Ä–µ–ø–∏—à–µ–º –æ–±–∞ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –≤ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–º –≤–∏–¥–µ** (–ø–æ —É–±—ã–≤–∞–Ω–∏—é —Å—Ç–µ–ø–µ–Ω–µ–π):

–ò—Å—Ö–æ–¥–Ω–æ–µ:
$$x^3 = \underbrace{0 \cdot x^2}_{\text{–Ω–µ—Ç } x^2} + \underbrace{3}_{\text{–∫–æ—ç—Ñ—Ñ. –ø—Ä–∏ } x} \cdot x + \underbrace{q}_{\text{—Å–≤–æ–±–æ–¥–Ω—ã–π —á–ª–µ–Ω}}$$

–ù–∞—à–µ:
$$x^3 = \underbrace{0 \cdot x^2}_{\text{–Ω–µ—Ç } x^2} + \underbrace{3uv}_{\text{–∫–æ—ç—Ñ—Ñ. –ø—Ä–∏ } x} \cdot x + \underbrace{u^3 + v^3}_{\text{—Å–≤–æ–±–æ–¥–Ω—ã–π —á–ª–µ–Ω}}$$

#### –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã

–ü–æ—Å–∫–æ–ª—å–∫—É –æ–±–µ —á–∞—Å—Ç–∏ —Ä–∞–≤–Ω—ã, **—Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã –¥–æ–ª–∂–Ω—ã —Å–æ–≤–ø–∞–¥–∞—Ç—å**:

**1) –ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –ø—Ä–∏ $x^2$:** 
$$0 = 0 \quad \checkmark \text{ (–≤—ã–ø–æ–ª–Ω–µ–Ω–æ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏)}$$

**2) –ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –ø—Ä–∏ $x^1$ (—Ç–æ –µ—Å—Ç—å –ø—Ä–∏ $x$):**
$$3uv = 3$$
$$\Rightarrow uv = 1$$

**3) –°–≤–æ–±–æ–¥–Ω—ã–π —á–ª–µ–Ω (–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –ø—Ä–∏ $x^0$):**
$$u^3 + v^3 = q$$

#### –ê–Ω–∞–ª–æ–≥–∏—è

–≠—Ç–æ –∫–∞–∫ –µ—Å–ª–∏ –±—ã —É –≤–∞—Å –±—ã–ª–æ:
$$5x + 7 = Ax + B$$

–ß—Ç–æ–±—ã —ç—Ç–æ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –±—ã–ª–æ –≤–µ—Ä–Ω–æ –ø—Ä–∏ **–ª—é–±–æ–º** $x$, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ:
- $A = 5$ (–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã –ø—Ä–∏ $x$ —Ä–∞–≤–Ω—ã)
- $B = 7$ (—Å–≤–æ–±–æ–¥–Ω—ã–µ —á–ª–µ–Ω—ã —Ä–∞–≤–Ω—ã)

### –®–∞–≥ 4: –°–∏—Å—Ç–µ–º–∞ —É—Ä–∞–≤–Ω–µ–Ω–∏–π

–ü–æ–ª—É—á–∞–µ–º —Å–∏—Å—Ç–µ–º—É –∏–∑ –¥–≤—É—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è $u$ –∏ $v$:

$$\begin{cases}
uv = 1 \\
u^3 + v^3 = q
\end{cases}$$

–ò–ª–∏ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ:

$$\begin{cases}
u^3 + v^3 = q \\
u^3 v^3 = 1
\end{cases}$$

(–≤—Ç–æ—Ä–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –ø–æ–ª—É—á–µ–Ω–æ –≤–æ–∑–≤–µ–¥–µ–Ω–∏–µ–º $uv = 1$ –≤ –∫—É–±)

### –®–∞–≥ 5: –†–µ—à–µ–Ω–∏–µ —Å–∏—Å—Ç–µ–º—ã

–û–±–æ–∑–Ω–∞—á–∏–º $t_1 = u^3$ –∏ $t_2 = v^3$. –¢–æ–≥–¥–∞:

$$\begin{cases}
t_1 + t_2 = q \\
t_1 \cdot t_2 = 1
\end{cases}$$

–≠—Ç–æ **—Ç–µ–æ—Ä–µ–º–∞ –í–∏–µ—Ç–∞**: $t_1$ –∏ $t_2$ —è–≤–ª—è—é—Ç—Å—è –∫–æ—Ä–Ω—è–º–∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è:

$$t^2 - qt + 1 = 0$$

–†–µ—à–∞—è, –ø–æ–ª—É—á–∞–µ–º:

$$t_{1,2} = \frac{q \pm \sqrt{q^2 - 4}}{2}$$

–°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ:

$$u = \sqrt[3]{\frac{q + \sqrt{q^2 - 4}}{2}}, \quad v = \sqrt[3]{\frac{q - \sqrt{q^2 - 4}}{2}}$$

–ò —Ä–µ—à–µ–Ω–∏–µ –∏—Å—Ö–æ–¥–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è:

$$x = u + v = \sqrt[3]{\frac{q + \sqrt{q^2 - 4}}{2}} + \sqrt[3]{\frac{q - \sqrt{q^2 - 4}}{2}}$$

## –ü–æ—á–µ–º—É –ø–æ–ª—É—á–∏–ª–∞—Å—å —Å–∏—Å—Ç–µ–º–∞ –∏–∑ –¥–≤—É—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π?

**–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:** –ú—ã –≤–≤–µ–ª–∏ **–¥–≤–µ –Ω–µ–∏–∑–≤–µ—Å—Ç–Ω—ã–µ** ($u$ –∏ $v$) –≤–º–µ—Å—Ç–æ –æ–¥–Ω–æ–π ($x$), –ø–æ—ç—Ç–æ–º—É –Ω—É–∂–Ω–æ **–¥–≤–∞ —É—Å–ª–æ–≤–∏—è** –¥–ª—è –∏—Ö –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è:

1. **–ü–µ—Ä–≤–æ–µ —É—Å–ª–æ–≤–∏–µ** ($uv = 1$) ‚Äî –º—ã –≤—ã–±–∏—Ä–∞–µ–º **–ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ**, —á—Ç–æ–±—ã –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã —Å–æ–≤–ø–∞–ª–∏
2. **–í—Ç–æ—Ä–æ–µ —É—Å–ª–æ–≤–∏–µ** ($u^3 + v^3 = q$) ‚Äî –≤–æ–∑–Ω–∏–∫–∞–µ—Ç –∏–∑ **—Å–≤–æ–±–æ–¥–Ω–æ–≥–æ —á–ª–µ–Ω–∞** –∏—Å—Ö–æ–¥–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è

–≠—Ç–æ –≥–µ–Ω–∏–∞–ª—å–Ω–∞—è –∏–¥–µ—è –ö–∞—Ä–¥–∞–Ω–æ: –¥–æ–±–∞–≤–∏—Ç—å —Å—Ç–µ–ø–µ–Ω—å —Å–≤–æ–±–æ–¥—ã (–≤—Ç–æ—Ä–æ–µ –Ω–µ–∏–∑–≤–µ—Å—Ç–Ω–æ–µ), –∞ –∑–∞—Ç–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ—ë –¥–ª—è —É–ø—Ä–æ—â–µ–Ω–∏—è –∑–∞–¥–∞—á–∏!

## –ü—Ä–∏–º–µ—Ä: –†–µ—à–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏—è $x^3 + 6x + 2 = 0$ –º–µ—Ç–æ–¥–æ–º –ö–∞—Ä–¥–∞–Ω–æ

### –®–∞–≥ 1: –û–ø—Ä–µ–¥–µ–ª—è–µ–º –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã

–£—Ä–∞–≤–Ω–µ–Ω–∏–µ –∏–º–µ–µ—Ç –≤–∏–¥ $x^3 + px + q = 0$, –≥–¥–µ:
- $p = 6$
- $q = 2$

### –®–∞–≥ 2: –°–æ—Å—Ç–∞–≤–ª—è–µ–º —Å–∏—Å—Ç–µ–º—É –¥–ª—è $u$ –∏ $v$

–ò—â–µ–º —Ä–µ—à–µ–Ω–∏–µ –≤ –≤–∏–¥–µ $x = u + v$, –≥–¥–µ $u$ –∏ $v$ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—é—Ç —Å–∏—Å—Ç–µ–º–µ:

$$\begin{cases}
3uv = p \\
u^3 + v^3 = -q
\end{cases}$$

**–ü–æ–¥—Å—Ç–∞–≤–ª—è–µ–º –∑–Ω–∞—á–µ–Ω–∏—è:**

$$\begin{cases}
3uv = 6 \\
u^3 + v^3 = -2
\end{cases}$$

**–£–ø—Ä–æ—â–∞–µ–º –ø–µ—Ä–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ:**

$$uv = 2$$

**–í–æ–∑–≤–æ–¥–∏–º –≤ –∫—É–±:**

$$u^3 v^3 = 8$$

**–°–∏—Å—Ç–µ–º–∞ –≤ —Ç–µ—Ä–º–∏–Ω–∞—Ö –∫—É–±–æ–≤:**

$$\begin{cases}
u^3 + v^3 = -2 \\
u^3 v^3 = 8
\end{cases}$$

### –®–∞–≥ 3: –ü—Ä–∏–º–µ–Ω—è–µ–º —Ç–µ–æ—Ä–µ–º—É –í–∏–µ—Ç–∞

–û–±–æ–∑–Ω–∞—á–∏–º $t_1 = u^3$, $t_2 = v^3$. –ü–æ —Ç–µ–æ—Ä–µ–º–µ –í–∏–µ—Ç–∞, –æ–Ω–∏ —è–≤–ª—è—é—Ç—Å—è –∫–æ—Ä–Ω—è–º–∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è:

$$t^2 - (t_1 + t_2)t + t_1 t_2 = 0$$

$$t^2 - (-2)t + 8 = 0$$

$$t^2 + 2t + 8 = 0$$

### –®–∞–≥ 4: –†–µ—à–∞–µ–º –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ

$$t = \frac{-2 \pm \sqrt{4 - 32}}{2} = \frac{-2 \pm \sqrt{-28}}{2}$$

**–ü–æ–ª—É—á–∏–ª–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –¥–∏—Å–∫—Ä–∏–º–∏–Ω–∞–Ω—Ç!** –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∫–æ—Ä–Ω–∏ –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã–µ. 

$$t = \frac{-2 \pm \sqrt{-28}}{2} = \frac{-2 \pm 2i\sqrt{7}}{2} = -1 \pm i\sqrt{7}$$

**–°—Ç–æ–ø!** –î–∞–≤–∞–π—Ç–µ –ø—Ä–æ–≤–µ—Ä–∏–º –∑–Ω–∞–∫–∏ –≤ —Å–∏—Å—Ç–µ–º–µ –µ—â–µ —Ä–∞–∑.

---

### –ò—Å–ø—Ä–∞–≤–ª–µ–Ω–∏–µ: –ø—Ä–∞–≤–∏–ª—å–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞

–î–ª—è —É—Ä–∞–≤–Ω–µ–Ω–∏—è $x^3 + 6x + 2 = 0$ (–±–µ–∑ –º–∏–Ω—É—Å–∞ –ø–µ—Ä–µ–¥ $3x$), –∏—Å–ø–æ–ª—å–∑—É–µ–º —Å–∏—Å—Ç–µ–º—É:

–ò–∑ $x^3 = u^3 + v^3 + 3uvx$ –∏ $x^3 = -6x - 2$:

$$\begin{cases}
3uv = -6 \quad \Rightarrow \quad uv = -2 \\
u^3 + v^3 = -2
\end{cases}$$

**–í–æ–∑–≤–æ–¥–∏–º $uv = -2$ –≤ –∫—É–±:**

$$u^3 v^3 = (-2)^3 = -8$$

**–°–∏—Å—Ç–µ–º–∞:**

$$\begin{cases}
u^3 + v^3 = -2 \\
u^3 v^3 = -8
\end{cases}$$

### –ü—Ä–∏–º–µ–Ω—è–µ–º —Ç–µ–æ—Ä–µ–º—É –í–∏–µ—Ç–∞

$u^3$ –∏ $v^3$ ‚Äî –∫–æ—Ä–Ω–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏—è:

$$t^2 - (-2)t + (-8) = 0$$

$$t^2 + 2t - 8 = 0$$

### –†–µ—à–∞–µ–º –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ

$$t = \frac{-2 \pm \sqrt{4 + 32}}{2} = \frac{-2 \pm \sqrt{36}}{2} = \frac{-2 \pm 6}{2}$$

$$t_1 = \frac{-2 + 6}{2} = 2, \quad t_2 = \frac{-2 - 6}{2} = -4$$

### –ù–∞—Ö–æ–¥–∏–º $u$ –∏ $v$

$$u^3 = 2 \quad \Rightarrow \quad u = \sqrt[3]{2}$$

$$v^3 = -4 \quad \Rightarrow \quad v = \sqrt[3]{-4} = -\sqrt[3]{4}$$

### –ù–∞—Ö–æ–¥–∏–º —Ä–µ—à–µ–Ω–∏–µ

$$\boxed{x = u + v = \sqrt[3]{2} - \sqrt[3]{4}}$$
