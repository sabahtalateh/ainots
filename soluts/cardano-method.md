# Метод Кардано для кубических уравнений

## Вывод формулы Кардано

Рассмотрим уравнение $x^3 - 3x - q = 0$ или $x^3 = 3x + q$.

### Почему $x = u + v$? Главная идея метода

**Вопрос:** Зачем искать решение в виде **суммы** двух неизвестных $x = u + v$?

**Ответ:** Это гениальная хитрость! Вот логика:

#### Проблема прямого решения

Кубическое уравнение $x^3 + px + q = 0$ имеет **одно неизвестное** $x$, но мы не можем выразить его напрямую (как в квадратном уравнении).

#### Идея Кардано: "Разделяй и властвуй"

**Вместо поиска одного $x$, будем искать два числа $u$ и $v$**, такие что их **сумма** даёт $x$:

$$x = u + v$$

**Зачем?** Теперь у нас **два неизвестных** вместо одного, но мы получаем **свободу выбора**: можем наложить на $u$ и $v$ любые **два условия**!

#### Магия формулы куба суммы

Когда мы подставляем $x = u + v$ в уравнение и используем формулу $(u+v)^3$, происходит чудо:

$$(u + v)^3 = u^3 + v^3 + 3uv(u + v)$$

Получается, что:
$$x^3 = u^3 + v^3 + 3uv \cdot x$$

**Главное наблюдение:** Кубическое уравнение для $x$ превращается в **два простых условия**:
1. Для линейного члена: $3uv = 3$ (коэффициент при $x$)
2. Для свободного члена: $u^3 + v^3 = q$

А эти два условия дают систему, которую можно решить через **квадратное уравнение** (используя теорему Виета)!

#### Аналогия

Представьте, что вы ищете число $x = 10$:
- **Прямой путь:** Решить сложное уравнение для $x$
- **Хитрость:** Найти два числа, например $u = 7$ и $v = 3$, чтобы $u + v = 10$

В случае кубического уравнения **прямого пути нет**, поэтому мы используем хитрость!

#### Почему именно сумма, а не произведение?

Можно было попробовать $x = uv$ (произведение), но:
- $(uv)^3 = u^3 v^3$ — нет полезной структуры
- $(u+v)^3 = u^3 + v^3 + 3uv(u+v)$ — **появляется линейный член** $3uv(u+v)$, который можно приравнять к $3x$ из исходного уравнения!

**Вывод:** Сумма даёт правильную алгебраическую структуру для работы с кубическим уравнением.

---

### Идея метода (кратко)

Будем искать решение в виде **суммы двух неизвестных**:

$$x = u + v$$

где $u$ и $v$ — пока неизвестные величины, которые мы подберем специальным образом.

### Шаг 1: Раскрытие куба суммы

Подставим $x = u + v$ в уравнение. Вспомним формулу куба суммы:

$$(u + v)^3 = u^3 + 3u^2v + 3uv^2 + v^3$$

Вынесем $3uv$ за скобку из средних членов:

$$(u + v)^3 = u^3 + v^3 + 3uv(u + v)$$

Поскольку $x = u + v$, получаем:

$$x^3 = u^3 + v^3 + 3uvx$$

### Шаг 2: Перестановка слагаемых

Перепишем в виде:

$$x^3 = 3uvx + (u^3 + v^3)$$

### Шаг 3: Метод неопределенных коэффициентов (сравнение коэффициентов)

У нас есть **два выражения для $x^3$**, которые должны быть равны:

**Исходное уравнение:**
$$x^3 = 3x + q$$

**Наше выражение (после подстановки $x = u + v$):**
$$x^3 = 3uvx + (u^3 + v^3)$$

#### Подробное объяснение метода сравнения коэффициентов

Если два многочлена равны **при любом** значении переменной, то **коэффициенты при одинаковых степенях** должны быть равны.

**Перепишем оба уравнения в стандартном виде** (по убыванию степеней):

Исходное:
$$x^3 = \underbrace{0 \cdot x^2}_{\text{нет } x^2} + \underbrace{3}_{\text{коэфф. при } x} \cdot x + \underbrace{q}_{\text{свободный член}}$$

Наше:
$$x^3 = \underbrace{0 \cdot x^2}_{\text{нет } x^2} + \underbrace{3uv}_{\text{коэфф. при } x} \cdot x + \underbrace{u^3 + v^3}_{\text{свободный член}}$$

#### Сравниваем коэффициенты

Поскольку обе части равны, **соответствующие коэффициенты должны совпадать**:

**1) Коэффициент при $x^2$:** 
$$0 = 0 \quad \checkmark \text{ (выполнено автоматически)}$$

**2) Коэффициент при $x^1$ (то есть при $x$):**
$$3uv = 3$$
$$\Rightarrow uv = 1$$

**3) Свободный член (коэффициент при $x^0$):**
$$u^3 + v^3 = q$$

#### Аналогия

Это как если бы у вас было:
$$5x + 7 = Ax + B$$

Чтобы это равенство было верно при **любом** $x$, необходимо:
- $A = 5$ (коэффициенты при $x$ равны)
- $B = 7$ (свободные члены равны)

### Шаг 4: Система уравнений

Получаем систему из двух уравнений для определения $u$ и $v$:

$$\begin{cases}
uv = 1 \\
u^3 + v^3 = q
\end{cases}$$

Или эквивалентно:

$$\begin{cases}
u^3 + v^3 = q \\
u^3 v^3 = 1
\end{cases}$$

(второе уравнение получено возведением $uv = 1$ в куб)

### Шаг 5: Решение системы

Обозначим $t_1 = u^3$ и $t_2 = v^3$. Тогда:

$$\begin{cases}
t_1 + t_2 = q \\
t_1 \cdot t_2 = 1
\end{cases}$$

Это **теорема Виета**: $t_1$ и $t_2$ являются корнями квадратного уравнения:

$$t^2 - qt + 1 = 0$$

Решая, получаем:

$$t_{1,2} = \frac{q \pm \sqrt{q^2 - 4}}{2}$$

Следовательно:

$$u = \sqrt[3]{\frac{q + \sqrt{q^2 - 4}}{2}}, \quad v = \sqrt[3]{\frac{q - \sqrt{q^2 - 4}}{2}}$$

И решение исходного уравнения:

$$x = u + v = \sqrt[3]{\frac{q + \sqrt{q^2 - 4}}{2}} + \sqrt[3]{\frac{q - \sqrt{q^2 - 4}}{2}}$$

## Почему получилась система из двух уравнений?

**Объяснение:** Мы ввели **две неизвестные** ($u$ и $v$) вместо одной ($x$), поэтому нужно **два условия** для их определения:

1. **Первое условие** ($uv = 1$) — мы выбираем **произвольно**, чтобы коэффициенты совпали
2. **Второе условие** ($u^3 + v^3 = q$) — возникает из **свободного члена** исходного уравнения

Это гениальная идея Кардано: добавить степень свободы (второе неизвестное), а затем использовать её для упрощения задачи!

## Резюме: Гениальность метода Кардано

### Схема решения

```
Кубическое уравнение для x (СЛОЖНО решить напрямую)
              ↓
    Подстановка x = u + v
              ↓
Формула куба суммы: (u+v)³ = u³ + v³ + 3uv(u+v)
              ↓
Сравнение коэффициентов → Система 2 уравнений для u³ и v³
              ↓
Теорема Виета → Квадратное уравнение (ЛЕГКО решить!)
              ↓
    Находим u и v → Находим x = u + v
```

### Ключевые моменты

1. **Понижение степени:** Кубическое уравнение → квадратное (через $u^3$ и $v^3$)
2. **Добавление свободы:** 1 неизвестное → 2 неизвестных, но это даёт гибкость
3. **Использование структуры:** Формула $(u+v)^3$ создаёт нужную алгебраическую форму
4. **Применение известной теории:** Теорема Виета связывает всё воедино

### Почему это важно

До XVI века **не существовало** общего метода решения кубических уравнений. Открытие метода Кардано (и его предшественника Тартальи) было революционным — оно показало, что сложные задачи можно решать через **хитрую подстановку** и **использование известных инструментов**.

Эта идея повлияла на всю алгебру: иногда **добавление сложности** (2 неизвестных вместо 1) **упрощает** решение!

## Пример: Решение уравнения $x^3 + 6x + 2 = 0$ методом Кардано

### Шаг 1: Определяем коэффициенты

Уравнение имеет вид $x^3 + px + q = 0$, где:
- $p = 6$
- $q = 2$

### Шаг 2: Составляем систему для $u$ и $v$

Ищем решение в виде $x = u + v$, где $u$ и $v$ удовлетворяют системе:

$$\begin{cases}
3uv = p \\
u^3 + v^3 = -q
\end{cases}$$

**Подставляем значения:**

$$\begin{cases}
3uv = 6 \\
u^3 + v^3 = -2
\end{cases}$$

**Упрощаем первое уравнение:**

$$uv = 2$$

**Возводим в куб:**

$$u^3 v^3 = 8$$

**Система в терминах кубов:**

$$\begin{cases}
u^3 + v^3 = -2 \\
u^3 v^3 = 8
\end{cases}$$

### Шаг 3: Применяем теорему Виета

Обозначим $t_1 = u^3$, $t_2 = v^3$. По теореме Виета, они являются корнями квадратного уравнения:

$$t^2 - (t_1 + t_2)t + t_1 t_2 = 0$$

$$t^2 - (-2)t + 8 = 0$$

$$t^2 + 2t + 8 = 0$$

### Шаг 4: Решаем квадратное уравнение

$$t = \frac{-2 \pm \sqrt{4 - 32}}{2} = \frac{-2 \pm \sqrt{-28}}{2}$$

**Получили отрицательный дискриминант!** Это означает, что корни комплексные. 

$$t = \frac{-2 \pm \sqrt{-28}}{2} = \frac{-2 \pm 2i\sqrt{7}}{2} = -1 \pm i\sqrt{7}$$

**Стоп!** Давайте проверим знаки в системе еще раз.

---

### Исправление: правильная система

Для уравнения $x^3 + 6x + 2 = 0$ (без минуса перед $3x$), используем систему:

Из $x^3 = u^3 + v^3 + 3uvx$ и $x^3 = -6x - 2$:

$$\begin{cases}
3uv = -6 \quad \Rightarrow \quad uv = -2 \\
u^3 + v^3 = -2
\end{cases}$$

**Возводим $uv = -2$ в куб:**

$$u^3 v^3 = (-2)^3 = -8$$

**Система:**

$$\begin{cases}
u^3 + v^3 = -2 \\
u^3 v^3 = -8
\end{cases}$$

### Применяем теорему Виета

$u^3$ и $v^3$ — корни уравнения:

$$t^2 - (-2)t + (-8) = 0$$

$$t^2 + 2t - 8 = 0$$

### Решаем квадратное уравнение

$$t = \frac{-2 \pm \sqrt{4 + 32}}{2} = \frac{-2 \pm \sqrt{36}}{2} = \frac{-2 \pm 6}{2}$$

$$t_1 = \frac{-2 + 6}{2} = 2, \quad t_2 = \frac{-2 - 6}{2} = -4$$

### Находим $u$ и $v$

$$u^3 = 2 \quad \Rightarrow \quad u = \sqrt[3]{2}$$

$$v^3 = -4 \quad \Rightarrow \quad v = \sqrt[3]{-4} = -\sqrt[3]{4}$$

### Находим решение

$$\boxed{x = u + v = \sqrt[3]{2} - \sqrt[3]{4}}$$

### Численная проверка

$$\sqrt[3]{2} \approx 1.2599$$

$$\sqrt[3]{4} \approx 1.5874$$

$$x \approx 1.2599 - 1.5874 \approx -0.3275$$

**Проверка:**

$$x^3 + 6x + 2 \approx (-0.3275)^3 + 6(-0.3275) + 2$$

$$\approx -0.0351 - 1.965 + 2 \approx -0.0001 \approx 0 \quad \checkmark$$

---

## Ответ

$$x = \sqrt[3]{2} - \sqrt[3]{4}$$

или в численном виде:

$$x \approx -0.3275$$
