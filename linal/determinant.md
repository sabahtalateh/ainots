## Аксиомы функции объёма $V$

| № | Название | Общее | 2D |
|:---:|---|---|---|
| 1 | Нормировка | $V(\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n) = 1$ <br> $V(\mathbf{e}_{p_1}, \mathbf{e}_{p_2}, \ldots, \mathbf{e}_{p_n}) = (-1)^{\operatorname{sign}(p)}$ | $V(\mathbf{e}_1, \mathbf{e}_2) = 1$ <br> $V(\mathbf{e}_2, \mathbf{e}_1) = -1$ |
| 2 | Обнуление | $V(\ldots, \mathbf{v}, \ldots, \mathbf{v}, \ldots) = 0$ | $V(\mathbf{v}, \mathbf{v}) = 0$ |
| 3 | Однородность | $V(\ldots, c\mathbf{v}, \ldots) = c\,V(\ldots, \mathbf{v}, \ldots)$ | $V(c\mathbf{v}, \mathbf{w}) = c\,V(\mathbf{v}, \mathbf{w})$ |
| 4 | Аддитивность | $V(\ldots, \mathbf{v} + \mathbf{w}, \ldots) = V(\ldots, \mathbf{v}, \ldots) + V(\ldots, \mathbf{w}, \ldots)$ | $V(\mathbf{v} + \mathbf{w}, \mathbf{u}) = V(\mathbf{v}, \mathbf{u}) + V(\mathbf{w}, \mathbf{u})$ |
| 5 | Кососимметричность | $V(\ldots, \mathbf{v}, \ldots, \mathbf{w}, \ldots) = -V(\ldots, \mathbf{w}, \ldots, \mathbf{v}, \ldots)$ | $V(\mathbf{v}, \mathbf{w}) = -V(\mathbf{w}, \mathbf{v})$ |

### Базисные векторы

$\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n$ — это стандартный базис пространства $\mathbb{R}^n$: каждый вектор $\mathbf{e}_i$ имеет $1$ на $i$-й позиции и $0$ на остальных. 

В **2D** $\mathbf{e}_1 = (1, 0)$, $\mathbf{e}_2 = (0, 1)$.

### Обнуление

При вычислении объёма/площади, если какая-то сторона, образованная вектором, равна другой стороне, площадь равна 0 (геометрически эти 2 стороны неотличимы друг от друга и являются одной и той-же стороной). Представьте прямоугольник, стороны которого всё ближе сходятся на одну линию, когда они полностью сойдутся, он превратится в линию, площадь которой будет равна 0. Так-же в 3D, если 2 из 3-х сторон сойдутся, у нас будет 2D фигура, объём которой будет равен 0.

**2D.** Два разных вектора $\mathbf{v}$, $\mathbf{w}$ натягивают параллелограмм. Когда $\mathbf{w}$ совпадает с $\mathbf{v}$, параллелограмм вырождается в отрезок — площадь равна 0:

```
V(v, w) ≠ 0           V(v, v) = 0

   *----*
  /░░░░/              *---------*
 /░░░░/               (отрезок, площадь = 0)
*----*
```

**3D.** Три разных вектора $\mathbf{v}$, $\mathbf{w}$, $\mathbf{u}$ натягивают параллелепипед. Когда $\mathbf{w}$ совпадает с $\mathbf{v}$, параллелепипед сплющивается в плоскую фигуру — объём равен 0:

```
V(v, w, u) ≠ 0            V(v, v, u) = 0

    *--------*                *--------*
   /|░░░░░░/|               /░░░░░░░░/
  / |░░░░░/ |              /░░░░░░░░/
 *--------* |             *--------*
 |  *-----|--*            (плоский, объём = 0)
 | /      | /
 |/       |/
 *--------*
```

### Однородность

Когда один из векторов умножается на число, то вся площадь/весь объём тоже умножается на это число.

**2D.** Умножение $\mathbf{v}$ на $c$ растягивает параллелограмм в $c$ раз:

```
   V(v, w) = A                  V(2v, w) = 2V(v, w) = 2A

   *------*                    *------+------*
  /░ A ░░/                    /░ A ░░/░ A ░░/
 /░░░░░░/                    /░░░░░░/░░░░░░/
*------*                    *------+------*
                             |← v →|← v →|
```

### Аддитивность

Параллелограмм на сумме векторов $\mathbf{v}+\mathbf{u}$ распадается на два отдельных параллелограмма:

```
V(v+u, w)                    V(v, w)  +  V(u, w)

   *----------*                  *-----*-----*
  /░░░░░░░░░░/                  /░░░░░/░░░░░/
 /░░░░░░░░░░/                  /░░░░░/░░░░░/
*----------*                  *-----*-----*
←── v+u ──→                   ←─v─→ ←─u─→
```

## Вывод формулы определителя

Раскладываем каждый вектор по базису и применяем правила 3–5:

$$V(\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_n)
= V\!\left(\sum_{i_1=1}^{n} v_{1i_1}\mathbf{e}_{i_1},\;
           \sum_{i_2=1}^{n} v_{2i_2}\mathbf{e}_{i_2},\;
           \ldots,\;
           \sum_{i_n=1}^{n} v_{ni_n}\mathbf{e}_{i_n}\right)$$

По правилу 4 (аддитивность) раскрываем все суммы:

$$= \sum_{i_1=1}^{n}\sum_{i_2=1}^{n}\cdots\sum_{i_n=1}^{n}
    V\!\left(v_{1i_1}\mathbf{e}_{i_1},\; v_{2i_2}\mathbf{e}_{i_2},\; \ldots,\; v_{ni_n}\mathbf{e}_{i_n}\right)$$

По правилу 3 (однородность) выносим скаляры:

$$= \sum_{i_1=1}^{n}\sum_{i_2=1}^{n}\cdots\sum_{i_n=1}^{n}
    v_{1i_1}\,v_{2i_2}\cdots v_{ni_n}\;
    V\!\left(\mathbf{e}_{i_1}, \mathbf{e}_{i_2}, \ldots, \mathbf{e}_{i_n}\right)$$

По правилу 2 слагаемые с совпадающими индексами равны нулю — остаются только перестановки:

$$= \sum_{\substack{p\;\text{— перестановка}\\\text{чисел}\;1,2,\ldots,n}}
    v_{1p_1}\,v_{2p_2}\cdots v_{np_n}\;
    V\!\left(\mathbf{e}_{p_1}, \mathbf{e}_{p_2}, \ldots, \mathbf{e}_{p_n}\right)$$

Подставляем правило 1:

$$\boxed{V(\mathbf{v}_1, \ldots, \mathbf{v}_n)
= \sum_{\substack{p\;\text{— перестановка}\\\text{чисел}\;1,2,\ldots,n}}
  (-1)^{\operatorname{sign}(p)}\,v_{1p_1}\,v_{2p_2}\cdots v_{np_n}}$$
