## Краткое описание

### [Теорема о пределе суммы](limsum.md)

**Формулировка:** Если $\lim_{n \to \infty} a_n = A$ и $\lim_{n \to \infty} b_n = B$, то:

$$\lim_{n \to \infty} (a_n + b_n) = A + B$$

Доказательство использует метод $\varepsilon$-$N$ и неравенство треугольника.

---

### [Теорема Гейне](heine.md)

**Формулировка:** Функция $f(x)$ имеет предел $L$ при $x \to a$ **тогда и только тогда**, когда для любой последовательности $\{x_n\}$ с $x_n \to a$ выполняется $f(x_n) \to L$:

$$\lim_{x \to a} f(x) = L \quad \Longleftrightarrow \quad \lim_{n \to \infty} f(x_n) = L$$

**Основное применение:** доказательство несуществования предела — достаточно найти две последовательности, дающие разные пределы.

**Пример:** Доказательство того, что $\lim_{x \to 0} \sin\frac{1}{x}$ не существует.

---

### [Теорема Пифагора](pythagoras.md)

**Формулировка:** В прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов:

$$c^2 = a^2 + b^2$$

Представлены несколько способов доказательства:
1. Через площади (метод Евклида)
2. Алгебраическое (через разложение квадрата)
3. Через подобие треугольников
4. Индийское (через перестановку)

Также включены: обобщения (теорема косинусов, многомерный случай), формула Евклида для пифагоровых троек, историческая справка.

---

### [Теорема Виета](viet.md)

**Формулировка:** Для квадратного уравнения $x^2 + px + q = 0$ с корнями $x_1$ и $x_2$:

$$\begin{cases}
x_1 + x_2 = -p \\
x_1 \cdot x_2 = q
\end{cases}$$

Представлены два способа доказательства:
1. Через разложение на множители и метод неопределённых коэффициентов
2. Через формулу корней квадратного уравнения

Также включены: обобщение для многочленов степени $n$, обратная теорема Виета, примеры применения и историческая справка.
