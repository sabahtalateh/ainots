# Теорема Пифагора

## Зачем нужна эта теорема?

Теорема Пифагора — один из **фундаментальных** результатов геометрии, связывающий стороны прямоугольного треугольника. Без неё невозможны:

- Вычисление расстояний на плоскости и в пространстве
- Тригонометрия (все тригонометрические тождества)
- Координатная геометрия Декарта
- Метод исчерпывания Архимеда (вычисление площадей многоугольников)

---

## Формулировка теоремы

**Теорема Пифагора:**

В прямоугольном треугольнике **квадрат гипотенузы равен сумме квадратов катетов**:

$$c^2 = a^2 + b^2$$

где $c$ — гипотенуза (сторона напротив прямого угла), $a$ и $b$ — катеты.

```
        /|
       / |
    c /  | b
     /   |
    /____|
      a
```

**Обратная теорема:**

Если в треугольнике выполняется равенство $c^2 = a^2 + b^2$, то треугольник **прямоугольный** (угол напротив стороны $c$ равен 90°).

---

## Доказательства теоремы

Существует более **400 различных доказательств** теоремы Пифагора! Рассмотрим несколько классических.

### Доказательство 1: Через площади (метод Евклида)

Это доказательство приведено в "Началах" Евклида (Книга I, Предложение 47).

**Идея:** Построим квадраты на всех сторонах треугольника и покажем, что площадь большого квадрата равна сумме площадей двух малых.

```
        ┌─────────┐
        │    c²   │
        │         │
        └─────────┘
             ↑
    ┌───┐   /\   ┌─────┐
    │a² │  /  \  │ b²  │
    │   │ /    \ │     │
    └───┘/______\└─────┘
```

**Построение:**

1. Дан прямоугольный треугольник ABC с прямым углом при C
2. Строим квадраты на всех сторонах: квадрат $a^2$ на катете $a$, квадрат $b^2$ на катете $b$, квадрат $c^2$ на гипотенузе $c$
3. Из вершины C проводим высоту на гипотенузу и продолжаем её до противоположной стороны квадрата на гипотенузе
4. Эта линия делит квадрат $c^2$ на два прямоугольника
5. Доказываем (через равенство треугольников), что:
   - Первый прямоугольник равен по площади квадрату $a^2$
   - Второй прямоугольник равен по площади квадрату $b^2$
6. Следовательно, $c^2 = a^2 + b^2$ ✓

---

### Доказательство 2: Алгебраическое (через разложение квадрата) ⭐

Это самое элегантное и наглядное доказательство.

**Построение:**

Возьмём квадрат со стороной $(a + b)$ и разместим внутри него 4 одинаковых прямоугольных треугольника с катетами $a$ и $b$:

```
    a       b
  ┌────┬────────┐
  │   /│        │ a
  │  / │   II   │
  │ /  │        │
  ├────┼────────┤
  │    │\       │
  │ I  │ \      │ b
  │    │  \     │
  └────┴────────┘
```

**Способ 1:** Треугольники по углам

```
  ┌─────────────┐
  │\     I     /│
  │ \         / │
  │  \_______/  │
  │  │       │  │
  │II│  c²   │IV│
  │  │_______│  │
  │  /       \  │
  │ /   III   \ │
  │/___________\│
  └─────────────┘
```

Площадь большого квадрата: $(a + b)^2$

Эта площадь состоит из:
- 4 треугольника с площадью $\frac{1}{2}ab$ каждый
- Квадрата в центре со стороной $c$ (гипотенуза)

**Уравнение:**

$$(a + b)^2 = 4 \cdot \frac{1}{2}ab + c^2$$

$$a^2 + 2ab + b^2 = 2ab + c^2$$

$$a^2 + b^2 = c^2 \quad ✓$$

---

### Доказательство 3: Через подобие треугольников

**Построение:**

1. Дан прямоугольный треугольник ABC с прямым углом при C
2. Опустим высоту CH из вершины C на гипотенузу AB

```
        C
       /|\
      / | \
   a /  |h \ b
    /   |   \
   /___ |____\
  A  m  H  n  B
       c = m + n
```

**Ключевое наблюдение:** Высота делит исходный треугольник на два меньших, и все три треугольника **подобны** друг другу!

- △ABC ~ △ACH ~ △CBH

**Из подобия:**

Треугольник ACH подобен ABC:
$$\frac{AC}{AB} = \frac{AH}{AC}$$
$$\frac{b}{c} = \frac{m}{b}$$
$$b^2 = mc$$

Треугольник CBH подобен ABC:
$$\frac{BC}{AB} = \frac{BH}{BC}$$
$$\frac{a}{c} = \frac{n}{a}$$
$$a^2 = nc$$

**Складываем:**
$$a^2 + b^2 = nc + mc = c(n + m) = c \cdot c = c^2 \quad ✓$$

---

### Доказательство 4: Индийское (через перестановку)

Это доказательство использовалось индийским математиком Бхаскарой II (XII век). Он сопровождал его одним словом: **"Смотри!"**

**Идея:** Разрежем два квадрата ($a^2$ и $b^2$) и соберём из них один квадрат ($c^2$).

```
Квадрат a²:        Квадрат b² (разрезанный на 4 части):
┌─────┐           ┌─────────┐
│     │           │\       /│
│     │           │ \     / │
│     │           │  \   /  │
│     │           │   \ /   │
│     │           │    X    │
└─────┘           │   / \   │
                  │  /   \  │
                  │ /     \ │
                  │/_______\│
                  └─────────┘

        ↓ Собираем в квадрат c² ↓

        ┌───────────────┐
        │\             /│
        │ \     ①    / │
        │  \_________/  │
        │  │         │  │
        │④ │   a²   │② │
        │  │_________│  │
        │  /         \  │
        │ /     ③    \ │
        │/_____________\│
        └───────────────┘
```

Четыре треугольника (вырезанных из квадрата $b^2$) и квадрат $a^2$ складываются в квадрат $c^2$.

## Как древние греки могли открыть теорему?

### Эмпирическое наблюдение

Египтяне и вавилоняне знали частные случаи задолго до доказательства:

**Египетский треугольник:** Стороны 3, 4, 5

$$3^2 + 4^2 = 9 + 16 = 25 = 5^2 \quad ✓$$

Строители использовали верёвку с 12 узлами (3 + 4 + 5 = 12) для построения прямых углов.

**Другие пифагоровы тройки:**
- (5, 12, 13): $25 + 144 = 169$ ✓
- (8, 15, 17): $64 + 225 = 289$ ✓
- (7, 24, 25): $49 + 576 = 625$ ✓

### Геометрическое открытие

Возможно, кто-то заметил, что для равнобедренного прямоугольного треугольника (с катетами 1 и 1):

```
    ┌───┐
    │   │\
    │ 1 │ \ √2
    │   │  \
    └───┴───┘
      1
```

Диагональ квадрата единичной площади **не выражается** целым числом или дробью! Это привело к открытию иррациональных чисел и глубокому изучению соотношения $1^2 + 1^2 = (\sqrt{2})^2$.

## Обобщения теоремы Пифагора

### Для произвольного треугольника (теорема косинусов)

$$c^2 = a^2 + b^2 - 2ab\cos C$$

При $C = 90°$: $\cos 90° = 0$, и получаем классическую теорему Пифагора.

### Для пространства

В трёхмерном пространстве для вектора с координатами $(x, y, z)$:

$$|v|^2 = x^2 + y^2 + z^2$$

### Для n-мерного пространства

$$|v|^2 = x_1^2 + x_2^2 + \ldots + x_n^2$$

## Применения

### 1. Вычисление расстояний

Расстояние между точками $(x_1, y_1)$ и $(x_2, y_2)$:

$$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

### 2. Высота равностороннего треугольника

Для равностороннего треугольника со стороной $a$:

```
       /\
      /  \
   a /    \ a
    /  h   \
   /___•____\
     a/2  a/2
```

По теореме Пифагора:
$$a^2 = h^2 + \left(\frac{a}{2}\right)^2$$
$$h^2 = a^2 - \frac{a^2}{4} = \frac{3a^2}{4}$$
$$h = \frac{a\sqrt{3}}{2}$$

### 3. Диагональ прямоугольника

$$d = \sqrt{a^2 + b^2}$$

### 4. Метод исчерпывания Архимеда

Архимед использовал теорему Пифагора для вычисления сторон вписанных и описанных многоугольников при оценке числа $\pi$ (см. [01-ancient.md](../histmatan/01-ancient.md)).

## Пифагоровы тройки

**Определение:** Тройка натуральных чисел $(a, b, c)$ называется **пифагоровой**, если $a^2 + b^2 = c^2$.

### Формула Евклида

Все примитивные пифагоровы тройки (где $\gcd(a, b, c) = 1$) задаются формулой:

$$a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2$$

где $m > n > 0$, $\gcd(m, n) = 1$, и $m - n$ нечётно.

**Примеры:**

| m | n | a | b | c | Проверка |
|---|---|---|---|---|----------|
| 2 | 1 | 3 | 4 | 5 | $9 + 16 = 25$ ✓ |
| 3 | 2 | 5 | 12 | 13 | $25 + 144 = 169$ ✓ |
| 4 | 1 | 15 | 8 | 17 | $225 + 64 = 289$ ✓ |
| 4 | 3 | 7 | 24 | 25 | $49 + 576 = 625$ ✓ |

## Историческая справка

### Пифагор Самосский (~570-495 до н.э.)

Теорема названа в честь древнегреческого философа и математика **Пифагора**, хотя она была известна и до него:

- **Вавилоняне** (около 1800 до н.э.) — знали частные случаи
- **Египтяне** — использовали треугольник 3-4-5 для строительства
- **Индия** (около 800 до н.э.) — упоминается в Шульба-сутрах

**Заслуга Пифагора:** Первое **дедуктивное доказательство** (а не просто наблюдение).

### Пифагорейский союз

Пифагор основал религиозно-философскую школу, члены которой:
- Считали числа основой мироздания
- Держали свои открытия в секрете
- Были потрясены открытием иррациональных чисел (диагональ единичного квадрата = $\sqrt{2}$)

По легенде, Гиппас из Метапонта был утоплен за разглашение тайны иррациональных чисел.
