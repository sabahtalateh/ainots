# Доказательства теорем

## Теорема о пределе суммы

**Теорема:** Если $\lim_{n \to \infty} a_n = A$ и $\lim_{n \to \infty} b_n = B$, то:

$$\lim_{n \to \infty} (a_n + b_n) = A + B$$

---

## Что значит "близко" и "стремится"?

Перед доказательством важно понять **точные** определения этих интуитивных слов.

### Бытовое понимание (неточное)

**"Близко"** — рядом, почти одинаково  
**"Стремится"** — приближается, становится всё ближе

Но в математике нужна **абсолютная точность**!

### Математическое определение "близко"

**"Число $x$ близко к числу $A$"** означает:

$$|x - A| < \varepsilon$$

где $\varepsilon$ (эпсилон) — это **насколько близко** мы хотим.

**Примеры:**

1. $x = 3.14$ близко к $\pi \approx 3.14159$ с точностью $\varepsilon = 0.01$?
   - $|3.14 - 3.14159| = 0.00159 < 0.01$ ✓ Да!

2. $x = 3.1$ близко к $\pi$ с точностью $\varepsilon = 0.001$?
   - $|3.1 - 3.14159| = 0.04159 > 0.001$ ✗ Нет!

**Ключевая идея:** "Близко" — это не абсолютное понятие, а **относительное**. Зависит от выбранного $\varepsilon$.

---

### Математическое определение "стремится"

**"Последовательность $a_n$ стремится к $A$"** означает:

> Для **любой** наперёд заданной точности $\varepsilon > 0$ найдётся такой номер $N$, что **все** члены последовательности с номерами больше $N$ находятся на расстоянии меньше $\varepsilon$ от $A$.

**Формально:**

$$\forall \varepsilon > 0 \quad \exists N: \quad \forall n > N \Rightarrow |a_n - A| < \varepsilon$$

**Читается так:**
- Для любого положительного $\varepsilon$ (как угодно малого!)
- Существует номер $N$ (начиная с которого...)
- Для всех $n > N$ выполняется $|a_n - A| < \varepsilon$ (все члены близки к $A$)

---

### Разбор по частям

#### 1. "Для любого $\varepsilon > 0$"

Это значит: **вы** выбираете точность. Хотите $\varepsilon = 0.1$? Пожалуйста. Хотите $\varepsilon = 0.000001$? Без проблем.

**Задача математика:** Показать, что для **любого** вашего выбора $\varepsilon$ можно найти $N$.

#### 2. "Существует номер $N$"

Это **ответ** математика на ваш выбор $\varepsilon$. Чем меньше $\varepsilon$, тем больше может быть $N$.

**Важно:** $N$ зависит от $\varepsilon$, часто пишут $N(\varepsilon)$.

#### 3. "Для всех $n > N$"

Начиная с номера $N$ и **дальше** — **все** члены последовательности (без исключений!) находятся в $\varepsilon$-окрестности от $A$.

---

### Наглядный пример

Рассмотрим $a_n = \frac{1}{n}$ и $A = 0$.

**Утверждение:** $\lim_{n \to \infty} \frac{1}{n} = 0$

**Что это значит:**

| Вы задаёте $\varepsilon$ | Я нахожу $N$ | Проверка |
|---|---|---|
| $\varepsilon = 0.1$ | $N = 10$ | Для $n > 10$: $\frac{1}{n} < 0.1$ ✓ |
| $\varepsilon = 0.01$ | $N = 100$ | Для $n > 100$: $\frac{1}{n} < 0.01$ ✓ |
| $\varepsilon = 0.001$ | $N = 1000$ | Для $n > 1000$: $\frac{1}{n} < 0.001$ ✓ |
| $\varepsilon$ любое | $N = \lceil \frac{1}{\varepsilon} \rceil$ | Для $n > N$: $\frac{1}{n} < \varepsilon$ ✓ |

**Вывод:** Как бы мало ни было $\varepsilon$, я всегда найду такой $N$, что после него все члены будут ближе к нулю, чем $\varepsilon$.

---

### Визуализация

```
             A = 0
             │
    ┌────────┼────────┐  ← "коридор" шириной 2ε
    │        │        │
────┴────────┼────────┴────→
ε   │        │        │  -ε
    │        │        │
    │  • •   │   ••••••  ← члены последовательности a_n
    │ •      │        
    •        │        
            
    ├────────┼────────┤
      N=1    N        
```

**Слева от $N$:** Члены могут быть где угодно  
**Справа от $N$:** Все члены внутри "коридора" $[-\varepsilon, \varepsilon]$

---

### Почему так сложно?

**Житейское "стремится":** "Ну, $\frac{1}{n}$ становится всё меньше..."

**Математическое "стремится":** "Для любой наперёд заданной точности можно гарантировать..."

Разница в **строгости**. Математическое определение:
- Не оставляет "лазеек"
- Работает для любых последовательностей
- Позволяет строго **доказывать** теоремы

## Интуитивное объяснение

### Что говорит теорема?

**Простыми словами:** Если две последовательности стремятся к своим пределам, то их сумма стремится к сумме пределов.

**Пример:**
- Последовательность $a_n = \frac{1}{n}$ стремится к $0$
- Последовательность $b_n = \frac{1}{n^2}$ стремится к $0$
- Тогда $(a_n + b_n) = \frac{1}{n} + \frac{1}{n^2}$ тоже стремится к $0 + 0 = 0$

### Почему это неочевидно?

На первый взгляд кажется: "Ну конечно, это же очевидно!" Но в математике нужно **строго доказать**, что:
- Если $a_n$ "почти равно" $A$
- И $b_n$ "почти равно" $B$
- То $(a_n + b_n)$ "почти равно" $(A + B)$

Проблема в том, что "почти равно" — это не точный термин. Нужно работать с формальным определением через $\varepsilon$ и $N$.

### Идея доказательства

**Задача:** Показать, что $(a_n + b_n)$ можно сделать как угодно близко к $(A + B)$.

**Стратегия:**

1. **Разбить задачу на части:**
   - Мы знаем, что $a_n$ близко к $A$
   - Мы знаем, что $b_n$ близко к $B$
   - Хотим показать, что $(a_n + b_n)$ близко к $(A + B)$

2. **Ключевое наблюдение:**
   
   $$|(a_n + b_n) - (A + B)| = |(a_n - A) + (b_n - B)|$$
   
   Расстояние от суммы до предела = расстояние первого слагаемого + расстояние второго слагаемого

3. **Трюк с $\frac{\varepsilon}{2}$:**
   - Нам нужно, чтобы вся сумма была меньше $\varepsilon$
   - Поэтому делим "бюджет погрешности" пополам:
     - $a_n$ отклоняется от $A$ не более чем на $\frac{\varepsilon}{2}$
     - $b_n$ отклоняется от $B$ не более чем на $\frac{\varepsilon}{2}$
   - В сумме получаем: отклонение не более $\frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$ ✓

4. **Синхронизация:**
   - $a_n$ становится близко к $A$ начиная с номера $N_1$
   - $b_n$ становится близко к $B$ начиная с номера $N_2$
   - Берём $N = \max(N_1, N_2)$ — тогда **одновременно** выполняются оба условия!

### Наглядная аналогия

Представьте:
- Вы идёте к точке $A$ (ваше расстояние до $A$ — это $|a_n - A|$)
- Ваш друг идёт к точке $B$ (его расстояние до $B$ — это $|b_n - B|$)
- Вопрос: на каком расстоянии от точки $(A, B)$ будет ваша пара?

**Ответ:** Ваше общее расстояние не больше суммы индивидуальных расстояний (это и есть неравенство треугольника).

Если вы оба приблизились к своим целям (на $\frac{\varepsilon}{2}$ каждый), то ваша пара приблизилась к общей цели (на $\varepsilon$).

---

## Формальное доказательство

Теперь переведём эту идею на строгий математический язык.

### Доказательство

**Дано:**
- $\lim_{n \to \infty} a_n = A$, то есть $\forall \varepsilon_1 > 0 \quad \exists N_1: \quad \forall n > N_1 \Rightarrow |a_n - A| < \varepsilon_1$
- $\lim_{n \to \infty} b_n = B$, то есть $\forall \varepsilon_2 > 0 \quad \exists N_2: \quad \forall n > N_2 \Rightarrow |b_n - B| < \varepsilon_2$

**Доказать:** 

$$\forall \varepsilon > 0 \quad \exists N: \quad \forall n > N \Rightarrow |(a_n + b_n) - (A + B)| < \varepsilon$$

**Доказательство:**

Пусть $\varepsilon > 0$ — произвольное положительное число.

Выберем $\varepsilon_1 = \varepsilon_2 = \frac{\varepsilon}{2}$.

Тогда:
- $\exists N_1$: для всех $n > N_1$ выполняется $|a_n - A| < \frac{\varepsilon}{2}$
- $\exists N_2$: для всех $n > N_2$ выполняется $|b_n - B| < \frac{\varepsilon}{2}$

Возьмем $N = \max(N_1, N_2)$.

Тогда для всех $n > N$ одновременно выполняются оба неравенства.

Рассмотрим:

$$|(a_n + b_n) - (A + B)| = |(a_n - A) + (b_n - B)|$$

По **неравенству треугольника** $|x + y| \leq |x| + |y|$:

$$|(a_n - A) + (b_n - B)| \leq |a_n - A| + |b_n - B|$$

Так как $n > N$, то:

$$|a_n - A| + |b_n - B| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

Следовательно:

$$|(a_n + b_n) - (A + B)| < \varepsilon$$

Это выполняется для любого $\varepsilon > 0$, следовательно, по определению:

$$\lim_{n \to \infty} (a_n + b_n) = A + B$$

**Что и требовалось доказать.** ∎

---

### Ключевые идеи

1. **Трюк с делением ε пополам** — берем $\frac{\varepsilon}{2}$ для каждой последовательности, чтобы их сумма дала $\varepsilon$
2. **Неравенство треугольника** — основной инструмент для оценки суммы модулей
3. **Максимум из N₁ и N₂** — обеспечивает одновременное выполнение обоих условий
