# Математический анализ

## Множество действительных чисел

### Аксиоматика

**Аксиоматический подход** — это метод построения математической теории, при котором:
1. Вводятся **аксиомы** — основные утверждения, принимаемые без доказательства
2. Из аксиом логически **выводятся следствия** (теоремы)
3. Вся теория строится на этом фундаменте

#### Определение

Непустое множество $\mathbb{R}$ называется **множеством действительных (вещественных) чисел**, а его элементы – **действительными (вещественными) числами**, если на $\mathbb{R}$ определены операции сложения и умножения и отношение порядка, удовлетворяющие следующим аксиомам.

#### 1°. Аксиомы сложения $(a, b \to a + b)$

1. $\forall a, b \in \mathbb{R} \to a + b = b + a$ (**коммутативность сложения**);

2. $\forall a, b, c \in \mathbb{R} \to a + (b + c) = (a + b) + c$ (**ассоциативность сложения**);

3. $\exists 0 \in \mathbb{R} : \forall a \in \mathbb{R} \to a + 0 = a$;

4. $\forall a \in \mathbb{R} \to \exists (-a) \in \mathbb{R}: a + (-a) = 0$ (число $(-a)$ называется **противоположным числом для $a$**).

#### 2°. Аксиомы умножения $(a, b \to ab)$

1. $\forall a, b \in \mathbb{R} \to ab = ba$ (**коммутативность умножения**);

2. $\forall a, b, c \in \mathbb{R} \to a(bc) = (ab)c$ (**ассоциативность умножения**);

3. $\exists 1 \in \mathbb{R}, 1 \neq 0 : \forall a \in \mathbb{R} \to 1 \cdot a = a$;

4. $\forall a \in \mathbb{R}, a \neq 0 \to \exists \frac{1}{a} \in \mathbb{R} : a \cdot \frac{1}{a} = 1$ (число $\frac{1}{a}$ называется **обратным числом для $a$**).

5. $\forall a, b, c \in \mathbb{R} \to a(b + c) = ab + ac$ (**дистрибутивность умножения относительно сложения**).

#### 3°. Аксиомы порядка (для любых $a, b \in \mathbb{R}$ установлено отношение $a \leqslant b$ или $b \leqslant a$)

1. $\forall a, b \in \mathbb{R} : a \leqslant b, b \leqslant a \to a = b$;

2. $\forall a, b, c \in \mathbb{R} : a \leqslant b, b \leqslant c \to a \leqslant c$;

3. $\forall a, b, c \in \mathbb{R} : a \leqslant b \to a + c \leqslant b + c$;

4. $\forall a, b \in \mathbb{R} : 0 \leqslant a, 0 \leqslant b \to 0 \leqslant ab$.

$a \leqslant b$ можно записывать также в виде $b \geqslant a$;

$a \leqslant b$ при $a \neq b$ можно записывать в виде $a < b$ или $b > a$.

#### 4°. Аксиома непрерывности (принцип Дедекинда)

Пусть $A, B$ – непустые подмножества $\mathbb{R}$ такие, что

$$\forall a \in A, b \in B \to a \leqslant b.$$

Тогда существует $c \in \mathbb{R}$ такое, что

$$\forall a \in A, b \in B \to a \leqslant c \leqslant b.$$

### Некоторые следствия из аксиом множества действительных чисел

#### 1. Число $0$ единственно

**Зачем это нужно:** Аксиома гарантирует **существование** нуля, но не гарантирует его **единственность**. Это доказательство показывает, что нейтральный элемент по сложению может быть только один — иначе арифметика была бы неоднозначной.

<details>
<summary>Доказательство</summary>

Предположим, что существует два нейтральных элемента по сложению: $0$ и $0'$, т.е.:
- $\forall a \in \mathbb{R}: a + 0 = a$ (по аксиоме 1.3)
- $\forall a \in \mathbb{R}: a + 0' = a$ (предположение)

Докажем, что $0 = 0'$. Рассмотрим сумму $0 + 0'$:
- С одной стороны, $0 + 0' = 0'$ (так как $0$ - нейтральный элемент)
- С другой стороны, $0 + 0' = 0$ (так как $0'$ - нейтральный элемент)

Следовательно: $0 = 0'$

Таким образом, нейтральный элемент единственен. ∎

</details>

#### 2. Для любого $a$ число $(-a)$, противоположное к $a$ единственно

**Зачем это нужно:** Для каждого числа может существовать только один противоположный элемент. Это гарантирует, что операция "взять противоположное число" определена однозначно, и мы не запутаемся, какое из нескольких возможных $(-a)$ выбрать.

<details>
<summary>Доказательство</summary>

Пусть для некоторого $a \in \mathbb{R}$ существует два противоположных элемента: $(-a)$ и $(-a)'$, т.е.:
- $a + (-a) = 0$ (по аксиоме 1.4)
- $a + (-a)' = 0$ (предположение)

Докажем, что $(-a) = (-a)'$. Чтобы "избавиться" от $a$ в левой части второго равенства и выделить $(-a)'$, прибавим к обеим частям элемент $(-a)$:

$$(a + (-a)') + (-a) = 0 + (-a)$$

По ассоциативности и коммутативности сложения:

$$(-a)' + (a + (-a)) = (-a)$$

Используя первое равенство $a + (-a) = 0$:

$$(-a)' + 0 = (-a)$$

$$(-a)' = (-a)$$

Таким образом, противоположный элемент единственен. ∎

</details>

#### 3. Для любых $a, b \in \mathbb{R}$ существует единственное $x$ такое, что $a + x = b$

**Зачем это нужно:** В аксиомах есть только сложение, но нет вычитания! Этот пункт показывает, что уравнение $a + x = b$ всегда разрешимо и имеет единственное решение, что позволяет **корректно определить операцию вычитания** как $b - a := b + (-a)$.

При этом $x = b + (-a)$; это число называется **разностью между $b$ и $a$** и обозначается $b - a$.

<details>
<summary>Доказательство</summary>

Докажем, что для любых $a, b \in \mathbb{R}$ уравнение $a + x = b$ имеет решение и это решение единственно.

*Существование.* Положим $x = b + (-a)$. Проверим, что $a + x = b$:

$$a + x = a + (b + (-a)) = a + ((-a) + b) = (a + (-a)) + b = 0 + b = b$$

Здесь мы использовали коммутативность и ассоциативность сложения.

*Единственность.* Пусть $a + x_1 = b$ и $a + x_2 = b$. Тогда:

$$a + x_1 = a + x_2$$

Прибавим к обеим частям $(-a)$:

$$(a + x_1) + (-a) = (a + x_2) + (-a)$$

По ассоциативности и коммутативности:

$$x_1 + (a + (-a)) = x_2 + (a + (-a))$$

$$x_1 + 0 = x_2 + 0$$

$$x_1 = x_2$$

Таким образом, решение уравнения $a + x = b$ существует и единственно. ∎

</details>

#### 4. Число $1$ единственно

**Зачем это нужно:** Аналогично пункту 1, но для умножения. Аксиома гарантирует **существование** единицы, а это доказательство показывает, что нейтральный элемент по умножению может быть только один.

<details>
<summary>Доказательство</summary>

Предположим, что существует два нейтральных элемента по умножению: $1$ и $1'$, т.е.:
- $\forall a \in \mathbb{R}: a \cdot 1 = a$ (по аксиоме 2.3)
- $\forall a \in \mathbb{R}: a \cdot 1' = a$ (предположение)

Докажем, что $1 = 1'$. Рассмотрим произведение $1 \cdot 1'$:
- С одной стороны, $1 \cdot 1' = 1'$ (так как $1$ - нейтральный элемент)
- С другой стороны, $1 \cdot 1' = 1$ (так как $1'$ - нейтральный элемент)

Следовательно: $1 = 1'$

Таким образом, нейтральный элемент по умножению единственен. ∎

</details>

#### 5. Для любого $a \neq 0$ число $\frac{1}{a}$, обратное к $a$ единственно

**Зачем это нужно:** Аналогично пункту 2, но для умножения. Для каждого ненулевого числа может существовать только один обратный элемент. Это гарантирует однозначность операции "взять обратное число".

<details>
<summary>Доказательство</summary>

Пусть для некоторого $a \in \mathbb{R}$, $a \neq 0$ существует два обратных элемента: $\frac{1}{a}$ и $\left(\frac{1}{a}\right)'$, т.е.:
- $a \cdot \frac{1}{a} = 1$ (по аксиоме 2.4)
- $a \cdot \left(\frac{1}{a}\right)' = 1$ (предположение)

Докажем, что $\frac{1}{a} = \left(\frac{1}{a}\right)'$. Чтобы "избавиться" от $a$ в левой части второго равенства и выделить $\left(\frac{1}{a}\right)'$, умножим обе части на элемент $\frac{1}{a}$:

$$\left(a \cdot \left(\frac{1}{a}\right)'\right) \cdot \frac{1}{a} = 1 \cdot \frac{1}{a}$$

По ассоциативности и коммутативности умножения:

$$\left(\frac{1}{a}\right)' \cdot \left(a \cdot \frac{1}{a}\right) = \frac{1}{a}$$

Используя первое равенство $a \cdot \frac{1}{a} = 1$:

$$\left(\frac{1}{a}\right)' \cdot 1 = \frac{1}{a}$$

$$\left(\frac{1}{a}\right)' = \frac{1}{a}$$

Таким образом, обратный элемент единственен. ∎

</details>

#### 6. Для любых $a \neq 0$ и $b$ существует единственное $x$ такое, что $ax = b$

**Зачем это нужно:** Аналогично пункту 3, но для умножения. В аксиомах нет деления! Этот пункт показывает, что уравнение $ax = b$ (при $a \neq 0$) всегда разрешимо и имеет единственное решение, что позволяет **корректно определить операцию деления** как $\frac{b}{a} := b \cdot \frac{1}{a}$.

При этом $x = b \cdot \frac{1}{a}$; это число называется **частным при делении $b$ на $a$** и обозначается $\frac{b}{a}$.

<details>
<summary>Доказательство</summary>

Докажем, что для любых $a \neq 0$ и $b \in \mathbb{R}$ уравнение $ax = b$ имеет решение и это решение единственно.

*Существование.* Положим $x = b \cdot \frac{1}{a}$. Проверим, что $ax = b$:

$$ax = a \cdot \left(b \cdot \frac{1}{a}\right) = a \cdot \left(\frac{1}{a} \cdot b\right) = \left(a \cdot \frac{1}{a}\right) \cdot b = 1 \cdot b = b$$

Здесь мы использовали коммутативность и ассоциативность умножения.

*Единственность.* Пусть $ax_1 = b$ и $ax_2 = b$. Тогда:

$$ax_1 = ax_2$$

Умножим обе части на $\frac{1}{a}$:

$$(ax_1) \cdot \frac{1}{a} = (ax_2) \cdot \frac{1}{a}$$

По ассоциативности и коммутативности:

$$x_1 \cdot \left(a \cdot \frac{1}{a}\right) = x_2 \cdot \left(a \cdot \frac{1}{a}\right)$$

$$x_1 \cdot 1 = x_2 \cdot 1$$

$$x_1 = x_2$$

Таким образом, решение уравнения $ax = b$ существует и единственно. ∎

</details>

#### 7. Для любого $a$ справедливо равенство $a \cdot 0 = 0$

**Зачем это нужно:** Это базовое свойство нуля при умножении. Оно не следует напрямую из аксиом (где говорится только о $a + 0 = a$), поэтому требует доказательства. Без этого свойства мы не могли бы утверждать, что умножение любого числа на ноль дает ноль.

<details>
<summary>Доказательство</summary>

**Идея доказательства:** Мы используем тот факт, что $0 + 0 = 0$, умножаем это равенство на $a$ и применяем дистрибутивность, чтобы получить $a \cdot 0 + a \cdot 0 = a \cdot 0$. Затем "сократим" одно $a \cdot 0$ с обеих сторон, прибавив противоположный элемент. Ключевое наблюдение: если $x + x = x$, то $x = 0$.

Докажем, что для любого $a \in \mathbb{R}$ выполняется $a \cdot 0 = 0$.

По аксиоме сложения: $0 + 0 = 0$.

Умножим обе части на $a$:

$$a \cdot (0 + 0) = a \cdot 0$$

По дистрибутивности (аксиома 2.5):

$$a \cdot 0 + a \cdot 0 = a \cdot 0$$

Прибавим к обеим частям $-(a \cdot 0)$:

$$(a \cdot 0 + a \cdot 0) + (-(a \cdot 0)) = a \cdot 0 + (-(a \cdot 0))$$

По ассоциативности:

$$a \cdot 0 + (a \cdot 0 + (-(a \cdot 0))) = 0$$

$$a \cdot 0 + 0 = 0$$

$$a \cdot 0 = 0$$

Таким образом, для любого $a$ выполняется $a \cdot 0 = 0$. ∎

</details>

#### 8. Пусть $a$ и $b$ такие, что $ab = 0$, тогда $a = 0$ или $b = 0$

**Зачем это нужно:** Это свойство показывает **отсутствие делителей нуля** в $\mathbb{R}$. Оно позволяет решать уравнения вида $ab = 0$ (раскладывая на множители) и гарантирует, что из $ab = ac$ и $a \neq 0$ следует $b = c$ (можно "сократить" на $a$). Это фундаментальное свойство полей.

<details>
<summary>Доказательство</summary>

**Идея доказательства:** Мы хотим доказать, что если произведение двух чисел равно нулю, то хотя бы один из сомножителей должен быть нулем (обратное к пункту 7). Используем разбор случаев: если $a \neq 0$, то умножим на $\frac{1}{a}$ и докажем, что $b = 0$.

Докажем, что если $ab = 0$, то $a = 0$ или $b = 0$.

Рассмотрим два случая:

**Случай 1:** $a = 0$. Тогда утверждение доказано.

**Случай 2:** $a \neq 0$. Тогда по аксиоме 2.4 существует обратный элемент $\frac{1}{a}$.

Умножим обе части равенства $ab = 0$ на $\frac{1}{a}$:

$$\frac{1}{a} \cdot (ab) = \frac{1}{a} \cdot 0$$

По ассоциативности умножения:

$$\left(\frac{1}{a} \cdot a\right) \cdot b = \frac{1}{a} \cdot 0$$

$$1 \cdot b = \frac{1}{a} \cdot 0$$

По пункту 7 имеем $\frac{1}{a} \cdot 0 = 0$, следовательно:

$$b = 0$$

Таким образом, в любом случае либо $a = 0$, либо $b = 0$. ∎

</details>

#### 9. Для любых $a$ и $b$ всегда имеет место одно и только одно из соотношений $a < b$, $a = b$, $a > b$

**Зачем это нужно:** Это свойство называется **законом трихотомии** (или законом сравнимости). Оно показывает, что любые два действительных числа можно сравнить, и результат сравнения однозначен: ровно одно из трех соотношений выполняется. Это фундаментальное свойство упорядоченности действительных чисел.

<details>
<summary>Доказательство</summary>

Докажем две части утверждения:

**Часть 1. Хотя бы одно из трех соотношений выполняется.**

По определению аксиом порядка, для любых $a, b \in \mathbb{R}$ установлено отношение $a \leqslant b$ или $b \leqslant a$ (или оба).

- Если $a \leqslant b$ и $b \leqslant a$, то по аксиоме 3.1 имеем $a = b$.
- Если $a \leqslant b$ и не $b \leqslant a$, то $a < b$.
- Если $b \leqslant a$ и не $a \leqslant b$, то $b < a$, т.е. $a > b$.

Таким образом, хотя бы одно из соотношений выполняется.

**Часть 2. Не более одного из трех соотношений выполняется.**

Докажем, что они попарно несовместимы:

- Если $a = b$, то не может быть $a < b$ (так как $a < b$ означает $a \leqslant b$ и $a \neq b$).
- Если $a = b$, то не может быть $a > b$ (аналогично).
- Если $a < b$, то $a \leqslant b$. Предположим $a > b$, т.е. $b < a$, тогда $b \leqslant a$. Из $a \leqslant b$ и $b \leqslant a$ следует $a = b$ (аксиома 3.1), что противоречит $a < b$ (так как $a < b$ означает $a \neq b$).

Следовательно, выполняется ровно одно из трех соотношений. ∎

</details>

#### 10. $1 > 0$

**Зачем это нужно:** Это утверждение устанавливает **ориентацию числовой прямой** и показывает, что единица (и, следовательно, все натуральные числа) являются положительными. Это фундаментальное свойство позволяет различать положительные и отрицательные числа и является основой для определения понятия "больше нуля".

<details>
<summary>Доказательство</summary>

**Идея:** По трихотомии либо $1 > 0$, либо $1 < 0$ (так как $1 \neq 0$). Предположим $1 < 0$ и получим противоречие.

---

Из аксиомы 2.3 знаем: $1 \neq 0$.

По закону трихотомии: либо $1 > 0$, либо $1 < 0$.

**Предположим $1 < 0$.**

Прибавим $(-1)$ к обеим частям $1 < 0$:
$$0 < -1$$

Значит, $-1 > 0$ (положительное число!).

По аксиоме 3.4, произведение двух положительных чисел неотрицательно:
$$(-1) \cdot (-1) \geqslant 0$$

Но $(-1) \cdot (-1) = 1$ (это можно проверить через дистрибутивность: $(-1)(1 + (-1)) = 0$, откуда $(-1) \cdot 1 + (-1) \cdot (-1) = 0$, т.е. $-1 + (-1) \cdot (-1) = 0$, поэтому $(-1) \cdot (-1) = 1$).

Получается: $1 \geqslant 0$.

Но мы предположили $1 < 0$. **Противоречие!**

Значит, $1 > 0$. ∎

</details>

#### 11. Пусть $a \leqslant b$, тогда $-b \leqslant -a$

**Зачем это нужно:** Это свойство показывает, что при переходе к противоположным числам **неравенство меняет направление**. Это важно для решения неравенств: когда мы умножаем неравенство на $-1$, знак неравенства переворачивается. Например, из $2 \leqslant 3$ следует $-3 \leqslant -2$.

<details>
<summary>Доказательство</summary>

**Идея:** Из $a \leqslant b$ получим $0 \leqslant b - a$, затем прибавим $-b$ к обеим частям и получим $-b \leqslant -a$.

---

Дано: $a \leqslant b$.

По аксиоме 3.3 прибавим к обеим частям $(-a)$:
$$a + (-a) \leqslant b + (-a)$$
$$0 \leqslant b - a$$

Теперь прибавим к обеим частям $(-b)$:
$$0 + (-b) \leqslant (b - a) + (-b)$$
$$-b \leqslant b + (-b) + (-a)$$
$$-b \leqslant 0 + (-a)$$
$$-b \leqslant -a$$

Таким образом, из $a \leqslant b$ следует $-b \leqslant -a$. ∎

</details>

### Примеры числовых множеств

- Множество **натуральных** чисел $\mathbb{N} = \{1, 2, 3, \ldots\}$, где $2 = 1 + 1$, $3 = 2 + 1$, $4 = 3 + 1$, $5 = 4 + 1, \ldots$

- Множество $\mathbb{N}_0 = \mathbb{N} \cup \{0\}$.

- Множество **целых** чисел $\mathbb{Z} = \{0, 1, -1, 2, -2, 3, -3, \ldots\}$.

- Множество **рациональных** чисел $\mathbb{Q} = \{x : x = \frac{m}{n}, m \in \mathbb{Z}, n \in \mathbb{N}\}$.

- Множество **иррациональных** чисел $\mathbb{R} \setminus \mathbb{Q}$.

## Верхние и нижние грани числовых множеств

### Определение

Множество $X \subset \mathbb{R}$ называется **ограниченным сверху**, если существует число $b$ такое, что

$$\forall x \in X \to x \leqslant b.$$

При этом говорят, что **число $b$ ограничивает множество $X$ сверху**.

---

Множество $X \subset \mathbb{R}$ называется **ограниченным снизу**, если существует число $a$ такое, что

$$\forall x \in X \to x \geqslant a.$$

При этом говорят, что **число $a$ ограничивает множество $X$ снизу**.

---

Множество $X \subset \mathbb{R}$ называется **ограниченным**, если оно ограничено и сверху, и снизу.

---

Множество $X \subset \mathbb{R}$ называется **неограниченным сверху**, если оно не является ограниченным сверху.

---

Множество $X \subset \mathbb{R}$ называется **неограниченным снизу**, если оно не является ограниченным снизу.

---

Множество $X \subset \mathbb{R}$ называется **неограниченным**, если оно не является ограниченным.

---

### Верхняя грань (супремум)

**Верхней гранью** непустого множества $X \subset \mathbb{R}$ называется число $b$, удовлетворяющее условиям:

1. $\forall x \in X \to x \leqslant b$;

2. $\forall b' < b \to \exists x \in X : x > b'$ 

   (эквивалентно: $\forall \varepsilon > 0 \to \exists x \in X : x > b - \varepsilon$).
