–ï—Å–ª–∏ $s(t)$ ‚Äî –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞ —Ç–µ–ª–∞, —Ç–æ $s'(t)$ ‚Äî **—Å–∫–æ—Ä–æ—Å—Ç—å**

### –ì–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏–π —Å–º—ã—Å–ª –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–π

$f'(x_0)$ ‚Äî **–Ω–∞–∫–ª–æ–Ω –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–π** –∫ –≥—Ä–∞—Ñ–∏–∫—É $y = f(x)$ –≤ —Ç–æ—á–∫–µ $(x_0, f(x_0))$

## –ü—Ä–∏–º–µ—Ä—ã –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã—Ö

### –ü—Ä–∏–º–µ—Ä 1: $f(x) = x^2$

$$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h} = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}$$

$$= \lim_{h \to 0} \frac{2xh + h^2}{h} = \lim_{h \to 0} (2x + h) = 2x$$

**–û—Ç–≤–µ—Ç:** $(x^2)' = 2x$

### –ü—Ä–∏–º–µ—Ä 2: $f(x) = x^3$

$$f'(x) = \lim_{h \to 0} \frac{(x+h)^3 - x^3}{h}$$

–†–∞—Å–∫—Ä—ã–≤–∞–µ–º $(x+h)^3 = x^3 + 3x^2h + 3xh^2 + h^3$:

$$= \lim_{h \to 0} \frac{x^3 + 3x^2h + 3xh^2 + h^3 - x^3}{h} = \lim_{h \to 0} \frac{3x^2h + 3xh^2 + h^3}{h}$$

$$= \lim_{h \to 0} (3x^2 + 3xh + h^2) = 3x^2$$

**–û—Ç–≤–µ—Ç:** $(x^3)' = 3x^2$

### –ü–∞—Ç—Ç–µ—Ä–Ω ‚≠ê

–ó–∞–º–µ—á–∞–µ–º –∑–∞–∫–æ–Ω–æ–º–µ—Ä–Ω–æ—Å—Ç—å:

$$(x^n)' = nx^{n-1}$$

–≠—Ç–æ **–ø—Ä–∞–≤–∏–ª–æ —Å—Ç–µ–ø–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏** (–¥–æ–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è –º–µ—Ç–æ–¥–æ–º –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–π –∏–Ω–¥—É–∫—Ü–∏–∏).

### 3. –û–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ –õ–µ–π–±–Ω–∏—Ü–∞ (–¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞–ª—å–Ω–∞—è —Ñ–æ—Ä–º–∞):
